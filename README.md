## 🚀 Getting started

Clone the project and access the folder

```bash
$ git clone git@github.com:damicolucas/art_museum.git && cd art_museum
```

Follow the steps below

```bash
# Install the dependencies
$ yarn install

# Start the project
$ yarn dev

```
